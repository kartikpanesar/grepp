#grepp

A Simple grep-like command-line tool written in C.

##Features
- Searches for pattern in text
- '-n': enables Line Numbers
- '-x': invert match

##Usage
- s1./grepp [options] pattern

##Examples
- ./grep -nx pattern
- ./grep -n -x pattern

##Future Improvements
- File Input support
- Case Insensitive search
- other options
